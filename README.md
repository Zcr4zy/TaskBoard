# TaskBoard
O projeto TaskBoard trata-se de uma web application simples de gerenciamento de tarefas, onde foi construído e estruturado com ASP.NET Web API e C# no back-end, e MVC com AJAX no front-end. Oferece funcionalidades de criação, edição e visualização de tarefas, permitindo uma organização simples e eficiente de atividades diárias.
